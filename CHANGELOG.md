# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-04-03

### Added
- Initial release of `animated-icons-vue`
- 402+ animated SVG icons for Vue 3
- Hover-triggered animations (translate, rotate, scale, opacity)
- Programmatic animation control via `startAnimation()` / `stopAnimation()`
- Props: `size`, `color`, `strokeWidth`, `active`, `className`
- Full TypeScript support
